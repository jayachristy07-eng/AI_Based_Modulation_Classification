# AI-Based Modulation Classification using Machine Learning

## Overview
This project implements an AI-based modulation classification system using Machine Learning. It classifies communication signals into BPSK, BFSK, and QPSK under noisy channel conditions.
The project uses FFT-based feature extraction and an MLP Neural Network for signal classification.
